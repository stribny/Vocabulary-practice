Dokumentace projektu =VocabularyPractice=
=========================================

Petr Stříbný, 12. prosince 2008

Zadání
------

: Řešil jsem úlohu pojmenovanou "Databáze slovíček". Cílem bylo vytvořit program za
: použití jazyka Haskell, který uživateli programu umožní procvičovat slovíčka
: cizích jazyků. Jednotlivá slovíčka, která by měla být definována v externím souboru,
: mají být vybírána náhodně.

Popis algoritmu
---------------

: Program můžeme rozdělit na dva logické celky. Ten první najde všechny dostupné slovníky
: (.txt soubory ve složce programu) a dá uživateli vybrat jaký slovník a v jakém směru 
: (česko-anglický nebo anglicko-český např.) chce procvičovat.

: Druhá část je pak smyčka, která neustále uživatele testuje na znalost slovní zásoby 
: (slovíčka jsou vybírána náhodně) dokud testování sám neukončí.

: Program je krátký a dobře dokumentovaný, věřím, že další popis není potřeba.

: Program neošetřuje chybné vstupy programu (prázdné řádky v souboru, atp.).

: Program byl vyzkoušen v interpretrech Hugs i GHCi.

Moduly
------

: Main --------	modul Main obsahuje funkci "main", která je hlavním vstupem programu
: Vocabulary --	tento modul obsahuje prakticky všechny funkce programu; neshledal jsem 
		totiž žádný důvod pro rozdělení programu do více modulů; program je stále
		přehledný

Formát vstupu
-------------

: Všechny .txt soubory, které se nachází v adresáři s programem jsou považovány za potenciální 
: vstupy (slovníky). Jejich seznam je k výběru uživateli nabídnut v průběhu programu, není tedy 
: zapotřebí předávat programu vstupní parametry. Soubor slovníku je organizován velice jednoduše:
: každý slovníkový záznam je na novém řádku a jednotlivá slova jsou oddělena středníkem. Součástí
: projektu jsou dva ukázkové vstupní soubory English.txt a Deutsch.txt.

Seznam souborů
--------------

: Main.hs ------------	modul Main se vstupní funkcí main
: Vocabulary.hs ------	modul Vocabulary, kde se nachází všechny ostatní funkce projektu
: English.txt --------	ukázkový vstupní soubor
: Deutsch.txt --------	ukázkový vstupní soubor
: dokumentace.txt ----	dokumentace projektu (právě čtete)
